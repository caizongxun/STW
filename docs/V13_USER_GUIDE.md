# V13 - DeepSeek-R1 AI 交易決策系統使用指南

## 🎯 系統特色

### 核心功能
1. **本地 AI 推理引擎**：使用 DeepSeek-R1 14B 進行完全本地化的交易決策
2. **Chain-of-Thought 推理**：展示完整的 AI 思考過程，提高可解釋性
3. **提示學習 (Prompt Learning)**：從歷史成功交易中學習，不斷優化決策
4. **多時間框架支援**：15m, 1h, 4h, 1d
5. **詳細信號展示**：精確的進場價、止損價、止盈價

## 📚 歷史數據學習機制

### 提示學習 (Prompt Learning) 原理

V13 不會修改 DeepSeek 模型本身，而是透過以下方式學習：

1. **記錄成功交易**：
   - 每次回測結束後，系統會自動保存「獲利 > 1%」的交易案例
   - 記錄內容包含：市場條件、AI 決策、實際結果

2. **注入學習上下文**：
   - 下次分析時，將這些成功案例注入到 Prompt 中
   - DeepSeek 會參考這些案例，學習在類似市場條件下的最佳決策

3. **持續優化**：
   - 學習庫最多保存最近 50 個成功案例
   - 每次回測都會更新學習庫

### 如何使用你的歷史數據進行學習

#### 方法 1：透過回測自動學習（推薦）

```python
# 在 GUI 中執行多次回測
# 系統會自動記錄成功案例到 data/success_cases.json
```

**步驟**：
1. 選擇幣種（例如 BTCUSDT）
2. 選擇時間框架（例如 15m）
3. 設定回測天數（建議 30-60 天）
4. 點擊「開始回測」
5. 系統會自動學習成功的交易

**查看學習進度**：
- 回測結束後，檢查 `data/success_cases.json` 文件
- GUI 底部會顯示「本次回測新增 X 個成功案例到 AI 學習庫」

## ⚙️ 進階設定

### 調整學習參數

編輯 `strategies/v13/config.py`：

```python
# 學習相關參數
enable_learning: bool = True  # 是否啟用學習
min_profit_to_learn: float = 0.01  # 最低獲利 1% 才記錄為成功案例
```

### 調整 AI 決策頻率

編輯 `strategies/v13/backtester.py`：

```python
# 每 N 根 K 線調用一次 AI（減少推理次數）
decision_interval = 4  # 15m * 4 = 每小時決策一次
```

如果你的電腦較快，可以改為：
```python
decision_interval = 1  # 每根 K 線都決策（更精確但更慢）
```

### 調整 AI 信心門檻

在 GUI 中透過滑桿調整，或在 `config.py` 中修改：

```python
ai_confidence_threshold: float = 0.70  # 70% 信心度才開倉
```

**建議值**：
- 保守策略：0.80 (只在 AI 非常有信心時開倉)
- 平衡策略：0.70 (預設)
- 積極策略：0.60 (較多交易機會但風險較高)

## ❓ 常見問題

### Q1: AI 推理速度很慢怎麼辦？

**A**: 有幾個優化方案：
1. 降低 `decision_interval`（減少決策頻率）
2. 使用量化模型：`ollama pull deepseek-r1:14b-q4_K_M`
3. 確認 GPU 驅動已安裝（`nvidia-smi`）

### Q2: 學習案例會一直累積嗎？

**A**: 不會，系統只保留最近 50 個成功案例，避免 Prompt 過長影響推理速度。

### Q3: 可以同時運行多個幣種的回測嗎？

**A**: 可以，但建議分開執行，因為 DeepSeek 推理會佔用 GPU 資源。

### Q4: 如何清空學習庫重新開始？

**A**: 刪除 `data/success_cases.json` 文件即可。

### Q5: V13 比 V12 好在哪裡？

**A**: 
- V12：使用 XGBoost，需要大量歷史數據訓練，推理較快
- V13：使用 DeepSeek-R1，具備推理能力，能理解市場邏輯，可解釋性更強

建議搭配使用：V12 用於量化信號，V13 用於最終決策驗證。

## 🚨 注意事項

1. **Ollama 必須運行**：確保後台有 Ollama 服務
2. **首次推理較慢**：模型需要加載到 VRAM，約 10-20 秒
3. **顯存需求**：DeepSeek-R1 14B 需要約 9GB VRAM
4. **學習需要時間**：至少進行 3-5 次回測才能累積有效案例
5. **風險提示**：AI 決策僅供參考，實盤前務必驗證
