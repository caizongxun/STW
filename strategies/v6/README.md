# V6 Strategy - 多时间框架融合与智能风控
## 特点
- 多时间框架融合：整合15m、1h、4h时间框架的特征，提升趋势识别能力
- 智能风控机制：动态止损、市况过滤，减少无效交易
- 双模型支持：XGBoost或LSTM深度学习模型可选
- 优化标签设计：提高最小收益阈值到1.5%，考虑实际交易成本
- 增强特征工程：添加K线形态（十字星、锤子线、包容线）、成交量特征（成交量突增、价量配合）

## 性能目标
- 月化报酬：20-30%
- 胜率：50-55%
- 利润因子：>1.5
- 最大回撤：<25%
- 月交易次数：50-100次

## 相比V5的改进
1. **标签质量提升**：最小收益阈值从0.8%提升到1.5%，并考虑0.2%的交易成本，确保信号的实际盈利性
2. **多时间框架特征**：新增1h和4h时间框架的RSI、MACD、SMA等特征，提升模型对中长期趋势的识别能力
3. **智能风控系统**：
   - 动态止损：基于ATR的动态止损，可调整止损倍数
   - 市况过滤：仅在趋势市场（1h MA20上穿/下穿MA50）生成信号，减少震荡市场的无效交易
4. **模型优化**：
   - 支持LSTM深度学习模型，提升时序预测能力
   - XGBoost模型参数可调整，降低过拟合风险
5. **特征工程增强**：
   - 新增K线形态特征：包容线、十字星、锤子线
   - 新增成交量特征：成交量突增、成交量趋势、价量相关性

## 开发规范
- 遵循项目的命名约定：类名使用PascalCase，函数使用snake_case
- 每个模块都有详细的注释，说明功能和参数
- 支持小数据集快速测试和完整数据集验证
